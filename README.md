# 🏠 House Price Prediction using Machine Learning

A machine learning project that predicts house prices based on various housing features such as bedrooms, bathrooms, square footage, location details, and other property attributes.

This project uses data preprocessing, feature engineering, and an XGBoost regression model to estimate house prices with high accuracy.

---

## 🚀 Features

- Data preprocessing and cleaning
- Handling missing values
- Correlation analysis with heatmaps
- Feature encoding for categorical data
- Train-test data splitting
- Model training using XGBoost Regressor
- Model evaluation using:
  - Mean Absolute Error (MAE)
  - R² Score
- House price prediction system
